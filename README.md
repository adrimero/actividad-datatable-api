# Actividad Práctica: Consumo de API y DataTables

**Autor:** Adrian Enrique Pincay Mero

## Descripción

Esta actividad tiene como objetivo desarrollar un sitio web estático utilizando tecnologías modernas de desarrollo frontend, integrando el framework de estilos TailwindCSS, el componente DataTables para la visualización de información tabular y el consumo de datos provenientes de una API mediante JavaScript asíncrono.

Durante el desarrollo se aplican conceptos fundamentales de control de versiones con Git y GitHub, manipulación del DOM, programación asíncrona con Promesas y `async/await`, procesamiento de datos JSON y despliegue de aplicaciones web estáticas en Vercel.

## Objetivos

- Crear y administrar un repositorio utilizando Git y GitHub.
- Documentar adecuadamente el proyecto mediante un archivo README.
- Implementar una interfaz web estática utilizando HTML, CSS y TailwindCSS.
- Integrar el componente DataTables para mostrar información de forma organizada.
- Consumir datos desde una API externa utilizando JavaScript moderno.
- Procesar y transformar los datos recibidos antes de mostrarlos al usuario.
- Aplicar buenas prácticas de organización de archivos y código.
- Desplegar la aplicación en un entorno de producción mediante Vercel.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- TailwindCSS
- DataTables
- Git
- GitHub
- Vercel
